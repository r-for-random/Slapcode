Slapcode
-
Slapcode is a comprehensive roleplay game framework I made to help my friends host roleplays on Roblox. It's free and open-source. At the moment, its features include:
- Character customization (R6 only)
- Naming
- Dice
- OOC

The system is built to be customized. There's a laundry list of things that developers can configure, from the types of dice players can use to the color and font of their names.

Slapcode's [Discord](https://discord.gg/xKhQ5EEq5e) hosts support, a changelog, suggestions and more.

Installation
-
Slapcode is exclusively developed using Roblox Studio, so it can only be downloaded as a .rbxm. To install it, download Slapcode.rbxm from the files above and insert it into ServerScriptService.
